## Node.js? 
JavaScript를 이용해서 웹 브라우저가 아닌 컴퓨터 자체를 제어한다. 
(파이썬, 루비, php 처럼!)
익숙한 JavaScript를 이용해서 웹 애플리케이션을 만들고 싶어진 사람들은 Node.js의 등장에 열광하게 되고, 웹 브라우저에서만 사용할 수 있었던 JavaScript가 핫한 컴퓨터 언어로 역전하게 됨

우리도 이 핫한 JavaScript를 이용해서 웹 애플리케이션을 만들어보자!
